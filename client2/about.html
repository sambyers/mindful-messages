What is Mindful Messages?

Mindful Messages is a Webex integration that provides a message scheduling service. It gives you an option to send a message to someone on the Webex platform at a time more in tune with their active hours. By scheduling a message for during someone's active hours, you are saving that person from a distracting notification and implicit expectations of working outside of their normal hours.

Demo POC Implementation Details

This deployment of Mindful Messages is a demo proof of concept. The goal of this demo is demonstrate integration with Webex using OAuth2, the Cisco UI Kit, webexteamssdk, and Epsagon. Other components in use are AWS Chalice, Lambda, API Gateway, and DynamoDB.

While you may get value out of using this demo, it is still a demo and comes with no support and new features and functionality will be added at the maintainers' leisure. Visit my website for a blog about what I learned making this or check out the code at GitHub.
